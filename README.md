# EMIS Grade Parser

Այս նախագիծը նախատեսված է [e-diary.emis.am](https://e-diary.emis.am/) կայքից վարժարանների/դպրոցների գնահատականները ավտոմատ կերպով հավաքելու (parsing), վերլուծելու և պահպանելու համար: Այն գործարկվում է հարմարավետ գրաֆիկական ինտերֆեյսով (GUI):

## Հիմնական Հնարավորություններ
* **Ավտոմատ մուտք համակարգ:** Աջակցում է Edge, Chrome և Brave բրաուզերներ:
* **Գնահատականների հաշվարկ:** 1-ին, 2-րդ կիսամյակների և տարեկան միջին/վերջնական գնահատականների ավտոմատ հաշվարկ:
* **Բացակայությունների հաշվառում:** Ամբողջական և կիսամյակային բացակայությունների գրանցում:
* **Թիրախային գնահատականի հաշվիչ:** Հաշվարկում է, թե որքան և ինչ գնահատականներ են անհրաժեշտ ցանկալի վերջնական արդյունքին հասնելու համար:
* **Արտահանում (Export):** Տվյալների արտահանում Excel (`.xlsx`) և հաշվետվությունների ստեղծում PDF ձևաչափով:

## Ինչպես Տեղադրել և Գործարկել

1. **Ներբեռնում:** Ներբեռնեք կամ կլոնավորեք այս նախագիծը ձեր համակարգչում:
2. **Python-ի տեղադրում:** Համոզվեք, որ ձեր համակարգչում տեղադրված է [Python](https://www.python.org/downloads/) (ցանկալի է 3.9 կամ ավելի նոր տարբերակ):
3. **Վիրտուալ միջավայր (ցանկալի է):** Խորհուրդ է տրվում ստեղծել և ակտիվացնել վիրտուալ միջավայր.
   ```bash
   python -m venv .venv
   
   # Windows PowerShell
   .venv\Scripts\Activate.ps1
   # կամ Windows Command Prompt
   .venv\Scripts\activate.bat
   ```
4. **Գրադարանների տեղադրում:** Տեղադրեք բոլոր անհրաժեշտ փաթեթները `requirements.txt` ֆայլից.
   ```bash
   pip install -r requirements.txt
   ```
5. **Գործարկում:** Միացրեք ծրագիրը հետևյալ հրամանով՝
   ```bash
   python main.py
   ```

## Օգտագործում
Գործարկելուց հետո կբացվի ծրագրի պատուհանը, որտեղ պետք է մուտքագրեք ձեր EMIS-ի մուտքանունը (Login) և գաղտնաբառը (Password), ընտրեք բրաուզերը և սեղմեք "Start Parsing": Տվյալները հավաքելուց հետո կարող եք տեսնել վիճակագրությունը վահանակի (Dashboard) վրա կամ արտահանել այն համապատասխան կոճակներով:
